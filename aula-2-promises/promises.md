# Ciclo de vida de promisses

Pending - Quando uma promise é instancianda ela vem com o status inicial de Pending( Pendente Ou ainda não terminou)
Fulfilled - Quando todas as operações são executadas com sucesso. 
Reject - Caso alguma coisa aconteceça de maneira inesperada ( usada para falhas )
